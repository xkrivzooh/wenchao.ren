出现这个异常的原因是因为我们的服务器使用的是java8，而工程代码中依赖的log4j-api-2.11.1.jar有部分java9版本的字节码文件。

然后排查了发现log4j-api-2.11.1.jar是通过elasticsearch的6.5.1引入的。
